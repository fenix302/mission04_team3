버스 이용 시

22, 102, G9633, 3, 75, 1-3, 1-1 : 버스정류장 GIDC (제B2813-1호) 하차

자동차 이용 시

서부 북부, 남부권
성산대교 북단 → 서부간선지하도로 진입 → 금천IC (시흥대교 방면) → 금하지하차도(소하분기점 방면) → 우회전 → 기아자동차 방면 좌회전 → 소하지하차도 진입 → 덕안터널 진입 → 롯데아울렛 광명점 앞 좌회전 → GIDC (제B2813-1호)

경기 시흥, 안산권
